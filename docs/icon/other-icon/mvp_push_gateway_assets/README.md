# MVP Push Gateway SVG/PNG Assets

这是一套基于“消息推送网关登录页”拆分的前端素材。

目录：
- svg/：可直接在 React/Vue/HTML 中使用的 SVG 文件。
- png/：由 SVG 导出的真透明 PNG，透明区域为 alpha=0，不是棋盘格假透明。

推荐用法：
- 页面主背景：background_light_tech_1920x1080.svg / .png
- 左侧完整插画：hero_routing_illustration_full.svg
- 路由核心：route_engine_core_3d.svg
- 登录卡右上角盾牌：icon_shield_verify_3d.svg
- 四个通道卡片：channel_sms_card、channel_mail_card、channel_im_card、channel_webhook_card
- 三个能力点：feature_rule_match、feature_template_render、feature_failed_retry
- 顶部品牌：header_logo_badges.svg 或 logo_paper_plane.svg + badge_*.svg

颜色主调：#1B70FF / #25B8FF / #EAF6FF / #102A55。

注意：带中文文字的 SVG 使用系统字体渲染，浏览器环境下会根据客户端字体显示。若要完全固定字体，需要在设计工具中转曲后再导出。
